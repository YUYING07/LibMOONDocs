# Installation
## Dependencies
To install `LibMOON`, we recommend to use the following libraries:

- Python == 3.11.9

## User Installation
- Create an environment`
```
    conde create -n libmoon python=3.11.9
    conda activate libmoon
    pip install libmoon==0.1.11    
```

- Clone the repository
```
    git clone https://github.com/xzhang2523/libmoon.git
```
